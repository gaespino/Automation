# Fuse File Generator

URL: `/thr-tools/fuse-generator`

## Purpose
Generate fuse configuration files from compute.csv and io.csv definitions.

## Usage
1. Select **Product**
2. Upload compute.csv and io.csv
3. Click **Load CSV Files**
4. Review fuse checklist
5. Click **Generate Fuse File**

## Backend
`THRTools/utils/fusefilegenerator.py`  `FuseFileGenerator` class
