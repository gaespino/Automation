# Automation Flow Designer

URL: `/thr-tools/automation-designer`

## Purpose
Build ordered step-list automation flow JSON files.

## Usage
1. Select **Step Type**
2. Enter **Step Label** and optional JSON **Parameters**
3. Click **Add Step**
4. Reorder with arrows; delete with trash icon
5. Click **Export** to download flow JSON
