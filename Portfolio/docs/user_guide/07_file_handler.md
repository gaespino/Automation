# File Handler

URL: `/thr-tools/file-handler`

## Purpose
Merge or append multiple Excel/CSV report files into a single output.

## Usage
1. Select operation: **Merge** or **Append**
2. Upload files
3. Set output filename
4. Click **Process Files**

## Backend
`THRTools/utils/PPVReportMerger.py`  `PPVReportMerger` class
