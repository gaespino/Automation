# DPMB Requests

URL: `/thr-tools/dpmb`

## Purpose
Submit and query DPMB requests programmatically.

## Usage
1. Select **Product**
2. Enter VIDs (one per line)
3. Set **Start WW** and **End WW**
4. Select **Operations**
5. Enter **Username**
6. Click **Submit Request** or **Status Check**

## Backend
`THRTools/api/dpmb.py`  `dpmb` class (non-GUI methods only)
