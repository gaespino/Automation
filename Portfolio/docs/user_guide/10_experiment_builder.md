# Experiment Builder

URL: `/thr-tools/experiment-builder`

## Purpose
Build multi-experiment JSON config files with per-product field enabling.

## Usage
1. Select **Product**  fields enabled per `{product}ControlPanelConfig.json`
2. Select **Environment**
3. Fill enabled fields
4. Click **Add Experiment**
5. Click **Export JSON**

## Import
Drag an existing JSON onto the import area to reload it.
